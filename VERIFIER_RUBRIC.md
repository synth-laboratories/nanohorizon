# ActorBench NanoHorizon Craftax Hello World Worker Rubric

Score the worker from `0.0` to `1.0`.

Criteria:

1. `artifact_bundle`
The rollout bundle exists.

2. `runner_success`
The result manifest records `status = succeeded`.

3. `contract_values`
The eval summary records the 10-rollout hello-world contract.

4. `reward_present`
The eval summary exposes the concrete reward.

5. `report_grounding`
The report cites the concrete run.
